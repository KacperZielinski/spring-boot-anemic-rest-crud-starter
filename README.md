# What is that?
Simple anemic model for typical REST CRUD applications.

### Stack
* OpenJDK 11
* Spring Boot 2.4.2
* Spring Data JPA
* H2
* Lombok

* Mockito

### Build
```mvn clean package -DskipTests```

### Test
```mvn test```

### Deploy
```mvn spring-boot:run```
